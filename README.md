# Todo List App

This project is a fully functional Todo List web application built using a vibe coding tool. The application allows users to manage daily tasks efficiently through a clean and simple interface.

## Features
- Add new tasks
- Edit existing tasks
- Mark tasks as completed
- Delete tasks
- Persistent storage using localStorage

## Technologies Used
- HTML
- CSS
- JavaScript

## Vibe Coding Tool
- Replit Agent (AI-powered coding assistant)

## Setup and Usage
1. Clone this repository.
2. Open the project folder.
3. Open the `index.html` file in your browser.
